はじめまして！


みるくてぃー？れもんてぃー？きゃのんてぃー！


9期研究生の最年長、きゃのんてぃこと石川花音です?


本題に行く前に、まずはプロフィールを！
(初回なので気合い入って長いですが、お付き合いください)




＊プロフィール＊


【名前】石川花音(いしかわかのん)
びっくりするくらい簡単な漢字です。笑






【ニックネーム】きゃのんてぃ
かのんって芸名でも結構多いんですよね。調べるといろんな方が出てきます！自分の名前好きだし、気に入ってるけど、単純にseo対策でもあるので、あえて変えてみました。
ミルクティー好きだし。笑
結構私の周りはきゃのんてぃって呼ぶ子、多いです！高校生の頃から呼ばれてるし、私もよく自分のこときゃのんてぃって言っちゃいます！笑
あ、でも全然他の呼び方でもウェルカムなので、自由に呼んで下さい?


コールの時呼ぶには長いですよね〜
そういう時は、「かのんー！」か、「のんてぃー！」がいいかな？って思っているのですが、いかがでしょうか？？






【出身地】千葉県
千葉県に住んでいましたが、高校は茨城県、大学は神奈川県、普段はよく東京都にいました！(忙しい)






【年齢】21歳
現在、大学3年生です！そして、松井珠理奈さんと同い年です！！






【身長】163cm






【好きな色】ピンク






【好きなこと】YouTube、インスタグラム、ツイッター、ブログ等のSNSを見ること&発信すること、動画編集、ファッション、メイク、音楽、踊ること


実は、SKE48に入る前にYouTubeやっていたんです！普段からご飯食べながら、お風呂入りながら、隙間時間にはすぐYouTubeを見てしまうくらい好きです！いろんなユーチューバーさんの動画を見ますが、私自身踊ってみた動画とかも上げていたので、今はパオパオチャンネルさんが1番好きです?(100万人おめでとうございます！)


ブログは、小・中学生くらいからやっていた気がします！もともと高校まで、びっくりするくらい、人前で喋るどころか、一対一でコミュニケーションを取ることすらもできなかったので、そのかわりにSNSを多用していました。なので、未だに人前に出るお仕事をしている自分が信じられませんっ！






【好きな食べ物】チョコ、クッキー、いちご、お寿司


チョコは中学生の頃から1日最低でも3個以上は必ず食べています。






【好きな飲み物】タピオカ、ミルクティー、カフェオレ


関東にしかないタピオカ屋さんも多いので、名古屋から帰った時は必ずどこかしら飲みに行っています！


ちなみにお酒はビールが1番好きです?






【性格】単純
褒められたら、調子に乗って、ぐんぐん伸びます！！！☆
だから、よかったらかのんのことたくさん褒めてあげてください←
それだけでhappyになれるので(^^)


先月、ボイトレがあったのですが、先生に褒めていただいて終わった後帰っているときもずっと一人でニコニコしてました( 怖い )


ボイトレ、すごく楽しかったです！先生とハイタッチもしました←


高校の頃、軽音楽部でガールズバンドを組んでいたのですが、ボーカルの子がものすごく歌が上手くて、ずっと憧れていたんです！
私も歌がうまくなりたい！って思って弾き語りするようになって、でも全然下手で、いつか習いたいって思ってたから嬉しかったな〜♪


それに対して、ダンスはまっっっっったく自信がありません！！！！！！(白目)


踊るのは好きなのですが、苦手なので、なかなか自信がないし、今とても行き詰まっています、、、


だから！もし！ほんの少しでも私のダンスがいいと思って頂けたら遠慮なく褒めてください！自信になります！！笑






【その他】車の免許持ってます！！(´-`)
いつかメンバー連れてどこか行きたいな〜

お付き合い頂きありがとうございます( ^ω^ )

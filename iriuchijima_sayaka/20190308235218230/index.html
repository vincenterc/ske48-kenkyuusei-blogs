今日はなんと！！
わたしの青春ガールズ公演！
劇場デビューでした！！






握手会、写メ会に足を運んできてくださってた方が沢山今日も来てくださってて嬉しかったです（ ; ; ）
わたしのファンの方が誰1人いなかったらどうしよう、って少し心配してたんです。(笑)






サイリウム白3本！
振ってくれた方ありがとうございました！
白3本って結構目立ちますね！それに決まって良かったです(^^)
わたしの写真が載ったTシャツやわたしの好きな某ひつじの写真が載ったさあや団扇、生写真を首に下げてくださってる方とか色々、、
最初から最後までバッチリ見てましたよ〜〜！幸せです。
それを見るたびに、さあやってコールを聞くたびに、すっごくすっごく嬉しくて笑みが溢れまくりでした。ごめんなさい。






わたしのコールですが、まだ完璧じゃないので今日披露できなかったです、、したかったのですが残念です(＞＜)
でも次は必ず披露したいので！！
完璧に決まったらまたブログに書くので練習お願いします（ ; ; ）

家族が見に来てくれて、2週間くらい会えずにレッスンしてたのですっごい寂しくて、帰りに会えたときのわたしの顔はグシャグシャでした(笑)
9期が出てきた瞬間、家族みんな感動したらしいんですよ。
それ聞いたときもですけどいつでもそれを考えただけでわたしも（ ; ; ）が。(笑)
入内嶋家は最高ですね〜離れて余計に大好きさが大っきくなりました。









青春ガールズの話に戻って



over ture って1番ドキドキの瞬間で、ああ始まる！！ってもう色々と凄かったです()

泣きそうになったのですが、必死に堪えて歌いました。






わたしは青春ガールズ、ビーチサンダルが1番好きです！



青春ガールズって全ての曲が楽しくて盛り上がれていいですよね！

でも



全てが初めてで完璧に出来ず、何してるの自分、、しか思わなかったです。
悔しい劇場デビューになりました。



スタッフさん、マネージャーさん、先輩方、9期の子、ドタバタでご迷惑お掛けしたことは反省してます。



今日言われたことを忘れずに、次回にそれを直して少しでも良くなったんじゃない？と言われるようにします。
それまでに先輩方のように必死に練習に取り組み、自分に厳しくしていきます。
言葉だけじゃなくてちゃんと行動するように。






初心の気持ちを忘れずにこれからもっともっと頑張らなきゃいけない、自分に危機感を1番痛感した日でした。






?

わたしは幸せ者ですね、応援してくださってる方にその分をこれから返していきます。



本当にありがとうございます。感謝してもしきれません！

◎生写真撮影


先日、生写真撮影をしました！


生写真撮影はアー写のとき以来で、
まだ２回目ですが前より少しだけ
緊張せずに撮ることができました！


いろんな衣装に着替えて
沢山写真を撮ってもらって
すっごく楽しかったです ⊂((・x・))⊃


どんな写真が使われるのか
今からでも楽しみです 見たいです。


19年 生きてきてちゃんとした服を着て
カメラマンさんに撮ってもらったのは
七五三以来の３回目です(笑)


小さい頃写真を撮られることが
すっごく苦手だったのですが、
今はやっと少しずつ慣れてきました(笑)


わたしは昔から写真映りが悪くて
実物と写真が全く違うとよく言われ、、
アー写なんか特にそうです。
あれは見るたびに悲しくなってます。
わたしと会ったことない方は、
まず写真で推すかも決めると思うのですが
一回会って見てから決めてもらえると
嬉しいです、、（ ;? ; ）笑
実際会ってみても
別に実物と写真変わらないじゃん！
って思って推さなくても大丈夫です(笑)




まだまだポーズをポンポンと変えて
撮影するということが慣れてなくて
同じようなポーズになってしまう事が
多いので先輩方の写真を見て勉強します。




今まで着たことのない衣装や、
一度は着たい！って思ってた衣装も
着ることが出来たので幸せです( ; _ ; )


わたしはSKE48の衣装で1番、
12月のカンガルーの衣装が好きなんです！
SHOWROOM中にも結構言ってたんですが
覚えてる方いますかね？？
あの可愛い衣装を着れる機会があったら
絶対に着たいです！！！
もうニヤニヤ止まらないですね〜〜
写真もたくさん撮っちゃうはずです。
家族にすぐに送って自慢します。


こういうことって考えただけで
本当に楽しいし着るために頑張ろう！
ってなるな〜〜（´-`）oO


発売されるかはまだ分かりませんが、
もし発売されることになったら
みなさん楽しみに待っていてください！
